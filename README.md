# BonBac ♻️

> Scanne un déchet, BonBac te dit dans quelle poubelle le jeter
> et où trouver le point de collecte le plus proche.

## Stack
Vue.js 3 · Node.js · PostgreSQL/PostGIS · TensorFlow.js · Leaflet

## Sources de données
- ADEME (points de collecte)
- Open Food Facts (produits)
- API Geo État (communes)
- data.gouv.fr
